
## 页面结构

### 1. 目录页 (index.html)
- 展示所有场景（大理 + 5个预留位置）
- 创作者入口

### 2. 场景体验页 (scene.html)
- 序章：场景描述 + 核心问题
- 三个核心入口：道士、观察者、路人总入口
- 点击「路人」→ 弹出角色选择 Modal
- 角色分类：游客(4) / 本地(1) / 定居者(3) / 元视角(1)

### 3. 创作者页 (creator.html)
- 创作理念
- 创作指南
- 叙事模板
- 投稿方式（预留）

## 如何使用

1. 打开 `index.html`
2. 选择场景进入
3. 阅读序章，选择视角
4. 输入你的思考，与角色对话
5. 随时切换视角

## 如何添加新叙事

1. 在 `experiences/` 记录真实观察
2. 在 `narratives/` 创建新模块
3. 在 `scripts/scenes.js` 中注册
4. 在 `scene.html` 中引入脚本
